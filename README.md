Astrbot 欢迎插件
✨ 简介
一款为 Astrbot 平台设计的欢迎新成员入群的插件，支持自定义欢迎模板、提示成员入群等功能。

🛠️ 功能特性
自定义欢迎模板
给与入群时间
提示入群用户自己的 UID
🎮 指令说明
🎯 欢迎开关指令
/welcome on
在群聊中使用，为本群开启自动欢迎新成员入群。

/welcome off
在群聊中使用，为本群关闭自动欢迎新成员入群。

⚙️ 设置打卡模板
/welcome set
在群聊中使用，为本群配置入群欢迎模板，在命令后面加模板。

支持{at}（@入群用户），{time}（入群时间），{user_id} （入群用户 UID）。

/welcome test
在群聊中使用，以自己为例，当前时间为准测试配置的模板。
